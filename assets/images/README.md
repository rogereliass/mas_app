# Image Assets

Place your image files in this directory:

## Required Images:

1. **mas_logo.png** - App logo (recommended size: 256x256px or higher)
2. **camp_30Y.jpg** - Hero/banner image for startup page (recommended size: 1200x800px or higher)

## Notes:
- Use PNG format for logos with transparency
- Use JPG format for photos to reduce file size
- Ensure images are optimized for mobile devices
- For different screen densities, you can create 2x and 3x versions

## After adding images:
Run `flutter pub get` to ensure assets are recognized, then hot reload or restart your app.
